How it works

One paste in. Four audiences out.

② Generate ~60 sec

Architecture

Next.js 15 · Gemini · Novus.ai — and it never hard-fails

Browser (App Router) /generate ─POST─▶ /api/report (Node runtime) │ validate (metrics + shipped) key set ──▶ Gemini 2.5 Flash ──▶ 4 formats (JSON) no key / error ──▶ buildDemoReport() deterministic fallback /generate ─sessionStorage─▶ /compare every action ─▶ Novus.ai / Pendo (report_generated, format_copied …)
The demo runs with zero secrets — no key just means deterministic output, never a broken screen.
